# AttendanceMgr
基于AngularJS的学生签到管理系统
## 教务考勤签到系统 ##
## 数据库设计说明
使用mongodb文档型数据库存储系统数据
### 实体说明：
- 学生：用于登录学生端，进行课程签到，保存学生个人信息
- 教师：用于登录教师端，进行学生、课程、考勤、选课等信息的管理，保存教师个人信息
- 课程：保存课程信息
- 签到次数：保存学生各种签到状态的总次数
- 签到记录：学生签到课程生成一条签到记录，保存学生签到状态、签到时间，以及课程部分信息
